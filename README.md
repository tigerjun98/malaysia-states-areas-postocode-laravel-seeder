# List of Malaysia States, Areas, Postcode and Location for Laravel Seeder.

## How to use
1. copy `Models/Malaysia` to `Models`.
2. copy `Seeder/LocationSeeder.php` to `database/Seeder`.
3. run `php artisan db:seed --class=LocationSeeder`
